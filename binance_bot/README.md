# 👹 Binance Scalping Monster Bot

이 봇은 바이낸스 선물(Futures) 시장에서 **SOL/USDT**를 대상으로 **초단타(Scalping)** 매매를 수행하는 고성능 봇입니다.

## ⚠️ 경고 (Warning)
- **높은 리스크**: 레버리지 7배와 단타 전략을 사용하므로 손실 위험이 있습니다.
- **자동 매매**: 봇이 실행 중일 때는 해당 계좌에서 수동 매매를 하지 마십시오. 자금 관리가 꼬일 수 있습니다.
- **테스트 필수**: 반드시 **테스트넷(Testnet)**에서 충분히 검증 후 실전에 투입하세요.

---

## 🚀 시작 가이드 (Quick Start)

### 1단계: 준비물 (Prerequisites)
1. **바이낸스 계정**: 선물(Futures) 계정이 활성화되어 있어야 합니다.
2. **API 키 (API Key)**:
   - [API Management] 페이지에서 키 발급.
   - **Enable Futures** 권한 체크 필수.
   - (보안) IP 제한을 설정하는 것을 권장합니다.
3. **Python**: 파이썬이 설치되어 있어야 합니다.

### 2단계: 설정 (Setup)
1. 폴더 내의 `.env.example` 파일을 복사하여 `.env` 로 이름을 변경합니다.
2. `.env` 파일을 메모장으로 열고 키를 입력합니다.
   ```ini
   BINANCE_API_KEY=내_API_키_붙여넣기
   BINANCE_SECRET_KEY=내_시크릿_키_붙여넣기
   BINANCE_TESTNET=True  <-- 처음엔 True로 테스트, 실전은 False
   ```

### 3단계: 설치 (Installation)
폴더 주소창에 `cmd`를 치고 아래 명령어를 입력하여 필요한 프로그램을 설치합니다.
```bash
pip install -r requirements.txt
```

### 4단계: 검증 (Verification)
봇이 거래소와 잘 연결되는지 확인합니다.
```bash
python test_connection.py
```
> ✅ 모든 항목이 초록색으로 뜨면 준비 완료입니다.

### 5단계: 실행 (Run)
`run_bot.bat` 파일을 더블 클릭하세요.
- **검은 창(터미널)**이 뜨고 로그가 올라오면 정상 작동 중입니다.
- 멈추려면 `Ctrl + C`를 누르거나 창을 닫으세요.

## 🧠 전략 설명 (Strategy logic)

### 1. 진입 (Entry)
- **종목**: SOL/USDT (변동성 우수)
- **타임프레임**: 3분봉 (3m)
- **Signal**: RSI + 볼린저 밴드 역매매
  - 🔻 **Short**: 가격이 상단 밴드 돌파 + RSI 과매수(70)
  - 🔺 **Long**: 가격이 하단 밴드 돌파 + RSI 과매도(30)
- **펀딩비 회피**: 01시, 09시, 17시 전후 15분간 진입 금지.

### 2. 청산 (Exit)
- **익절 (Take Profit)**:
  - **1차 (Split)**: 수익률 +1.0% 도달 시 **50% 물량 매도**. 남은 물량은 본절(Entry Price)로 스탑 로스 이동.
  - **2차 (Trail)**: 수익률 +1.5% 도달 시 **트레일링 스탑** 발동 (고점 대비 -0.5% 하락 시 전량 매도).
- **손절 (Stop Loss)**:
  - 진입과 동시에 **-0.8%** 지점에 강제 스탑 주문(Server Order) 등록.

### 3. 안전 장치 (Safety)
- **고스트 체크**: 주문 전 항상 실제 포지션을 재확인하여 중복 주문 방지.
- **더스트 정리**: 포지션 종료 시 미체결 주문(잔해) 자동 취소.
